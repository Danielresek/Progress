## Overview

Progress is a full-stack workout tracking application designed to help users structure workouts, log exercises, and monitor training progress over time.

The project was built as a personal project to explore modern full-stack development, authentication, and cloud deployment.

Users can create workout plans, log training sessions, and track their strength progress in a simple and mobile-friendly interface.

---

## Features

- Secure user authentication with Auth0
- Create custom workout plans
- Log sets, reps, and weights
- Track workout history
- Mobile-friendly interface
- Installable as a mobile app (PWA)

---

## Architecture

The application follows a simple full-stack architecture:

Frontend  
React + TypeScript application built with Vite.

Backend  
ASP.NET Core API responsible for workout plans, workout logs, and user data.

Database  
PostgreSQL database hosted on Render.

Authentication  
Auth0 handles user authentication with Google login.
